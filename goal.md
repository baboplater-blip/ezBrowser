# goal.md — 무엇을 완성하려는가

> 이 문서는 **변하지 않는 목적지**다. "무엇을 만들지"가 흔들릴 때 여기로 돌아온다.
> 계획(어떻게)은 [plan.md](plan.md), 현재 위치는 [status.md](status.md), 불가침 제약은 [rules.md](rules.md), 합격 기준은 [test.md](test.md).

---

## 한 문장

**가볍고, 크롬 확장이 그대로 붙고, 콕콕처럼 처음부터 쓸 만하고, 그 어떤 브라우저보다 자유로운 데스크톱 브라우저를 — 낯선 일반 사용자가 설치해서 주력 브라우저로 삼을 만큼 완성한다.**

지금까지(묶음 A~FF)는 **기능을 만드는 일**이었다. 이제부터는 **그 기능들을 낯선 사람 손에 쥐여줘도 무너지지 않는 제품으로 마감하는 일**이다.

---

## "완성품"의 정의 — 이 5가지가 모두 참이어야 한다

1. **설치가 된다.** 처음 보는 사람이 설치 파일을 받아 더블클릭하면, 경고에 막히지 않고(서명됨), 5분 안에 자기 브라우저로 쓰기 시작한다.
2. **안 죽는다.** 하루 종일 켜놓고 수십 개 탭을 열어도 크래시·흰 화면·먹통이 없다. 죽어도 다음 실행에 그대로 복원된다.
3. **첫인상이 좋다.** 첫 실행에서 무엇을 하는 브라우저인지 알게 되고, 크롬에서 북마크·비밀번호·기록을 가져오며, 기본 설정만으로 광고가 막히고 동영상이 받아진다.
4. **믿을 수 있다.** 비밀번호·방문기록·쿠키가 안전하게 다뤄지고(로컬·암호화), 무엇을 수집하지 않는지 명확하며, 라이선스 의무를 지킨다.
5. **갱신된다.** 버그를 고치면 자동 업데이트로 사용자에게 닿는다. 사용자가 재설치할 필요가 없다.

이 5개 중 **하나라도 거짓이면 "완성품"이 아니다.** 기능이 아무리 많아도 마찬가지다.

## 누구를 위한 것인가 (대상 사용자)

| 1순위 | Cốc Cốc/크롬을 쓰는 일반 사용자 | 광고 차단·동영상 다운로드·번역이 **기본**으로 되는 가벼운 브라우저를 원한다. 설정을 만지지 않는다. |
| 2순위 | 브라우저를 길들이고 싶은 파워 유저 | userChrome·userscript·정책 엔진·워크스페이스로 **자기 환경을 만든다.** "Emacs of browsers". |
| 3순위 | 한국·베트남 사용자 | 한국어/베트남어 UI, 한국 광고 필터(List-KR), 현지 검색엔진·뉴스·날씨가 기본. |

**1순위가 떠나면 실패다.** 자유도(2순위)는 1순위의 가벼움·안정성을 절대 해치지 않는 선에서만 존재한다.

## 성공의 측정 (Definition of Done — 출시 가능 상태)

아래가 전부 충족되면 **v1.0 출시 가능**으로 본다. 측정 방법은 [test.md](test.md).

### A. 출시 가능성
- [ ] Windows 코드 서명된 NSIS 인스톨러 (SmartScreen 경고 없음 또는 평판 축적 경로 확보)
- [ ] 자동 업데이트가 실제 저장소(GitHub Releases `owner/repo` 확정)에 연결되어 동작
- [ ] 첫 실행 온보딩 (브라우저 소개 + 크롬 데이터 가져오기 + 기본 브라우저 설정 유도)
- [ ] 크롬/엣지에서 북마크·비밀번호·방문기록·열린 탭 **가져오기(import)**

### B. 안정성 (실사용 검증)
- [ ] 50탭 + 8시간 연속 사용 세션에서 크래시 0, 흰 화면 0, 메모리 누수 없음
- [ ] 비정상 종료 후 재시작 시 탭·그룹·분할·스크롤·세션 100% 복원
- [ ] 알려진 회귀 패턴(#1~#14) 재현 테스트 전부 통과
- [ ] 콘솔/로그에 unhandled 에러 0

### C. 성능 예산 (1원칙 #1, 실측)
- [ ] 콜드 스타트 ≤ 2.0s · 빈 창 RSS ≤ 250MB · 탭당 ≤ 80MB · 휴식 CPU ≤ 0.5%
- [ ] 외피 초기 JS ≤ 500KB gzip (현재 ~65KB, 여유 충분)
- [ ] `/audit-perf` 자동 측정이 회귀를 막는다

### D. 핵심 약속 실증
- [ ] 콕콕 기본 기능 14종 전부 실사용 동작 (adblock·동영상·토렌트·번역·스크린샷·…)
- [ ] 동영상 다운로드 다사이트 매트릭스 통과 (progressive/HLS/DASH/토큰CDN/blob)
- [ ] 크롬 확장 호환 매트릭스 10종 로드·기본 동작 (uBO Lite·Dark Reader·Bitwarden·Vimium·…)
- [ ] 자유도 7축 + 자유도 모듈 12종 전부 사용 가능

### E. 신뢰
- [ ] 개인정보 처리방침 + 무엇을 수집/전송하는지 문서화 (기본: 수집 0, 외부 전송은 검색 제안·번역·위젯뿐)
- [ ] OSS 라이선스 고지 페이지 (`browser://licenses`) 자동 생성·최신
- [ ] 보안 기본값(rules.md) 전 세션·전 partition 강제 검증

### F. 접근·현지화
- [ ] 한국어·영어 100% 번역 (i18n 키 누락 0), 베트남어 핵심 화면
- [ ] 기본 키보드 접근성 (모든 핵심 동작 키맵 도달 가능 — 이미 actionId 체계 보유)

### G. 정체성
- [ ] 제품 이름·앱 아이콘·브랜드 컬러 확정 (현재 임시 "BrowserBuild")
- [ ] 랜딩 페이지 또는 배포 채널 (다운로드 받을 곳)

## 북극성 지표 (North Star)

> **"설치한 사람이 7일 뒤에도 이걸로 웹을 본다."**

기능 개수가 아니라 **잔존(retention)**이 척도다. 한 명이 설치하고 기본 브라우저로 바꾸고 일주일 뒤에도 쓰고 있으면, 이 제품은 일하고 있는 것이다. 모든 plan.md의 작업은 이 한 줄에 기여하는지로 우선순위를 매긴다.
