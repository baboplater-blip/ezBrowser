# ezBrowser 설치 안내

가볍고 자유로운 Chromium 기반 브라우저입니다. 광고 차단·동영상 다운로드·번역·워크스페이스 등이 기본 내장돼 있습니다.

## 시스템 요구 사항

- **Windows 10 / 11 (64비트)**
- 디스크 약 300MB

## 설치 방법

1. **`ezBrowser-0.1.0-win-x64.exe`** 를 다운로드합니다.
2. 다운로드한 파일을 **더블클릭**합니다.
3. **파란색 경고 화면이 뜹니다** — *"Windows의 PC를 보호했습니다"*. 아래 순서로 넘어갑니다:
   - **`추가 정보`** 를 클릭 →
   - 아래에 나타나는 **`실행`** 버튼 클릭
4. 설치 위치를 묻지 않고 **자동으로 설치되고, 끝나면 ezBrowser가 바로 실행됩니다.**

설치가 끝나면 **바탕화면**과 **시작 메뉴**에 `ezBrowser` 바로가기가 생깁니다.

---

## 왜 경고가 뜨나요? (안전한가요?)

이 경고는 **바이러스 경고가 아닙니다.** ezBrowser가 아직 **코드 서명 인증서(유료)** 로 서명되지 않아서, Windows가 "처음 보는 게시자"라고 표시하는 것뿐입니다. 개인·오픈소스 프로그램이 초기에 흔히 겪는 정상적인 화면입니다.

찝찝하시면 아래 **체크섬 검증**으로 파일이 변조되지 않았는지 직접 확인할 수 있습니다.

### 파일 무결성 검증 (선택)

PowerShell을 열고 다운로드한 파일 경로로 아래를 실행합니다:

```powershell
Get-FileHash .\ezBrowser-0.1.0-win-x64.exe -Algorithm SHA256
```

출력된 해시가 배포처에 게시된 값과 같으면 파일이 원본 그대로라는 뜻입니다.

**이 빌드(0.1.0)의 SHA256:**

```
48EBCE9BD900B60EF7C8B63B8D5E89E67F3DBDD633A38F35792CBAA256AA9179
```

> 새 버전이 나올 때마다 해시는 바뀝니다. 각 릴리스의 정확한 해시는 그 릴리스 배포처에 함께 게시됩니다.

## 설치 위치

- 프로그램: `%LOCALAPPDATA%\Programs\browser-build\`
- 사용자 데이터(북마크·이력·설정): `%APPDATA%\browser-build\`
- **관리자 권한(UAC)이 필요 없습니다** — 사용자 단위로 설치됩니다.

## 제거

- **설정 → 앱 → 설치된 앱** 에서 `ezBrowser` → 제거, 또는
- 시작 메뉴에서 `ezBrowser` 폴더의 제거 항목 실행

## 커스텀 위치 / 포터블로 쓰고 싶다면

설치 없이 쓰려면 `win-unpacked` 폴더(빌드 산출물)를 원하는 위치에 통째로 복사한 뒤 그 안의 `ezBrowser.exe` 를 직접 실행하면 됩니다.
