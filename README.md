# ezBrowser

가볍고, 크롬 확장이 그대로 붙고, 자유도가 높은 데스크톱 브라우저. 내장 AI 에이전트가 로그인된 사이트에서 사용자가 원하는 작업을 대신 수행합니다.

## 설치

[Releases](https://github.com/baboplater-blip/ezBrowser/releases) 에서 최신 `ezBrowser-<버전>-win-x64.exe` 를 내려받아 실행하세요. (미서명 빌드라 첫 실행 시 Windows SmartScreen 경고가 뜰 수 있습니다 — "추가 정보 → 실행".)

## 자동 업데이트

설치 후에는 앱이 새 버전을 자동으로 확인합니다. 새 버전이 나오면 앱 안에 업데이트 알림이 뜨고, 클릭하면 재시작하며 최신 버전으로 교체됩니다. (설정 → 업데이트 에서 수동 확인·채널 변경 가능)
