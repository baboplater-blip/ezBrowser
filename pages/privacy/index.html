개인정보 처리방침

ezBrowser · 최종 수정일 2026-06-21

요약: ezBrowser 는 당신을 추적하지 않습니다.
방문 기록·북마크·비밀번호·설정 등 모든 개인 데이터는 오직 이 컴퓨터에만 저장됩니다. 우리(개발자)에게 전송되는 데이터는 없습니다. 사용 통계·텔레메트리·광고 식별자를 수집하지 않습니다.

1. 로컬에만 저장되는 데이터

아래 데이터는 당신의 컴퓨터(사용자 데이터 폴더)에만 보관되며 외부로 나가지 않습니다.

  • 방문 기록·북마크·읽기 목록·다운로드 목록
  • 설정·단축키·워크스페이스·탭 그룹·세션(열린 탭)
  • 저장된 비밀번호 — 운영체제의 보안 저장소(Windows DPAPI / macOS Keychain / Linux libsecret)로 암호화되어 저장됩니다. 평문으로 디스크에 남지 않습니다.
  • 쿠키·사이트 데이터 — 일반 브라우저와 동일하게 사이트별로 저장되며, 자물쇠 메뉴·설정에서 삭제할 수 있습니다.

2. 외부로 나가는 통신 (기능에 필요한 최소한)

아래는 해당 기능을 사용할 때만 발생하는 외부 요청입니다. 각 기능은 설정에서 끌 수 있습니다.

기능전송 대상전송 내용
검색어 자동완성선택한 검색엔진(Google/Naver/Bing 등)주소창에 입력 중인 글자
웹 검색선택한 검색엔진검색어 (일반 브라우저와 동일)

이 외에 개발자가 운영하는 분석 서버나 광고 네트워크로의 전송은 없습니다. 위 서드파티 서비스는 각자의 개인정보 정책을 따릅니다.

3. 광고·트래커 차단

ezBrowser 는 기본적으로 광고와 추적기를 차단합니다(EasyList·EasyPrivacy·한국 KR 필터). 이는 당신을 추적하려는 제3자 스크립트를 막아 오히려 개인정보를 보호합니다.

4. 시크릿 창

시크릿 창의 방문 기록·쿠키·세션은 메모리에만 보관되며 창을 닫으면 삭제됩니다.

5. 데이터 내보내기·삭제

모든 데이터는 당신의 것입니다. 설정 → 데이터 관리에서 전체를 내보내거나 가져올 수 있고, 방문 기록·사이트 데이터는 언제든 삭제할 수 있습니다.

6. 문의

이 방침은 제품 변경에 따라 갱신될 수 있으며, 변경 시 이 페이지에 반영됩니다.
